## 你能获得什么？

- 审核通过后会获得 **$200 赠金**，可直接用于 Poixe 平台内的模型 API 调用抵扣。
- 访问平台提供的 **[全部模型资源](/cn/docs/models-pricing/models)**，按需选择不同模型来做课程演示、科研验证或项目开发。
- 使用平台支持的 **[4 种主流接口协议](/cn/api-reference/introduction/protocols)**，既能用标准对话接口快速接入，也能覆盖 **[Claude Code](/cn/use-cases/code/claude-code)** 等工具类客户端的调用方式。
- 可以接入市面上大部分开源项目与客户端。只要项目支持自定义 API 接口，就可以很方便地对接到 Poixe。

## 申请条件（简要）

我们主要核对四件事：

1. **组织真实**：确实来自高校场景（课程/实验室/社团/研究组等）
2. **用途清晰**：用于教学、科研、课程项目、社团活动或相关 demo
3. **信息可验证**：提供基本的学校/组织信息与联系人方式（建议学校邮箱）
4. **账号已注册**：请提前注册 Poixe 账号，并在申请表中填写该账号邮箱；我们将把赠金发放到该账号，并通过该邮箱通知审核结果

<Callout type="warning" icon="info-circle">
  为避免滥用：资料造假、批量注册、转售/代充/灰产用途等将直接拒绝，并可能限制账号。
</Callout>
