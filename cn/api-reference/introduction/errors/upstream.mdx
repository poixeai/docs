title: "上游错误"
sidebarTitle: "上游错误"
---

上游错误来自模型厂商或其网关。它通常意味着：请求已经被转发到上游，并由上游返回了错误响应。

## 常见原因

- 请求参数不符合协议要求（字段缺失、类型错误、取值非法）
- 鉴权失败（上游 Key/权限问题）
- 上游限流或过载
- 上游服务异常或维护

## 如何识别

查看响应头中的来源标识：

```text
Poixe-Error-Source: upstream
Poixe-Request-Id: <request_id>
```

同时，上游错误的响应体一般会保持上游协议的原始错误结构（可能因协议不同而不同）。

## 处理建议

* 如果是参数错误：优先对照对应协议的文档页检查请求体字段
* 如果是限流/过载：建议在客户端实现退避重试（并降低并发）
* 如果是偶发性 5xx：可重试；若持续出现，携带 Request ID 联系团队
