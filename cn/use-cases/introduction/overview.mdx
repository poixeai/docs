Poixe AI 提供统一且规范化的 API 接口协议，可将多种主流 AI 应用无缝接入到你的工作流或产品中。  

你通常只需要两样东西：`base_url` 和 `api_key` ——把它们填到对应的“接口配置”里，就能直接开始对话或调用。

本节将介绍几个常见的应用场景与接入方式，助你快速上手。
